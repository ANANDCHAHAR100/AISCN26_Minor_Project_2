# AISCN'26 Minor Project 2 – Web Application Security Assessment

## 📌 Project Overview

This project was completed as part of the AISCN'26 Cybersecurity Internship Programme. The objective was to understand web application security by performing reconnaissance, traffic analysis, and security assessment using OWASP Juice Shop and Burp Suite.

---

## 🎯 Objectives

- Understand the architecture of a web application.
- Perform reconnaissance on OWASP Juice Shop.
- Capture and analyze HTTP requests using Burp Suite.
- Study HTTP History and application behaviour.
- Generate and verify MD5 and SHA hash values.
- Learn bug bounty concepts and web security fundamentals.

---

## 🛠️ Tools Used

- OWASP Juice Shop
- Burp Suite Community Edition
- Web Browser
- MD5 & SHA Hash Generators

---

## 🔍 Activities Performed

- Explored the OWASP Juice Shop application.
- Intercepted HTTP requests and responses.
- Analysed HTTP History.
- Observed application endpoints.
- Generated MD5 and SHA hashes.
- Documented findings with screenshots.

---

## 📚 Learning Outcomes

Through this project, I gained practical exposure to:

- Web Application Security
- Burp Suite workflow
- HTTP request/response analysis
- Reconnaissance techniques
- Hashing concepts
- Bug Bounty fundamentals
